## 项目简介

一套专为审计场景设计的数据分析工具集，帮助审计人员快速完成大规模财务数据的清洗、分析和可视化。

## 核心功能

- **数据清洗**：自动识别并处理异常值、缺失值、重复数据
- **异常检测**：基于统计方法和机器学习的异常交易识别
- **趋势分析**：多维度财务指标趋势分析和同比环比
- **可视化报表**：自动生成交互式分析报告

## 技术栈

- Python + Pandas + NumPy
- Streamlit（Web 界面）
- Plotly（交互式图表）
- scikit-learn（异常检测模型）
