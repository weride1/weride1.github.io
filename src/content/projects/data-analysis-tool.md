## 项目背景

审计过程中需要大量数据分析工作：账账核对、账实核对、异常交易识别等。这些工作重复性高，适合工具化。

## 核心功能

- **账务核对**：自动比对两套账务数据，高亮差异项
- **Benford 分析**：首位数字定律检测，快速发现异常分布
- **趋势分析**：按期间、科目、部门多维度趋势展示
- **批量处理**：支持 Excel/CSV 批量导入导出

## 技术亮点

使用 Streamlit 构建 Web 界面，非技术人员也能轻松使用。
